This R program reads in information about peak predictions, coordinates, and Zoonomia genomes to produce visualizations follow-up analysis on TACIT applied to vocal learning

Peak Predictions for Mouse PV: V_MoHu_AvgActivityPredictions_boreoeutheria.txt (Available from Kaplow et. al. 2023)
Peak Predictions for Mouse M1: MoMaRaBaEnhancersNRSummit.txt (Available from Kaplow et. al. 2023)

basicGenomeQualStats.2.1.1.v2simp.csv - Genome Quality Stats From Zoonomia Project (Christmas et. al. 2023, provided here)
200Mammal_hal_tree_noancestors_matrix.csv - Species Evolutionary Histories (Christmas et. al. 2023, provided here)
Zoonomia_ChrX_lessGC40_241species_30Consensus.tree - Mammalian Evolutionary Tree (Christmas et. al. 2023, provided here)
Zoonomia-Hiller_species_name_conversions.csv - Naming conversion for Zoonomia Species (Kirilenko et. al. 2023, provided here)

data/Boreoeutheria_VLtrait_MW.csv - Vocal Learning Trait Annotations

cortex_vl_results_combined_mousefix_12-31_atLeast3True_orthologInChiropteraVocalLearner_orthologInNonChiropteraVocalLearner_adjp.csv - Motor Cortex TACIT vocal learning results
pv_vl_results_combined_atLeast3True_orthologInChiropteraVocalLearner_orthologInNonChiropteraVocalLearner_adjp.csv - PV+ inhibitory interneuron TACIT results

/projects/pfenninggroup/machineLearningForComputationalBiology/Cortex_Cell-TACIT/data/raw_data/filtered_peaks_final/BICCN_Mouse/ - Directory containing BED files for single nucleus ATAC-Seq peaks across cortical neural cell types (A multimodal cell census and atlas of the mammalian primary motor cortex, 2021)
